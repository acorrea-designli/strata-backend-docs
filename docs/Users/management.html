User management

Change password​

mutation Mutation($data: SetNewPasswordInput!) {
  setNewPassword(data: $data)
}
{
  "data": {
    "user": {
      "uuid": "" // user uuid
    },
    "password": "" // new password
  }
}

Set user active/inactive​

mutation UpdateUser($data: UpdateUserInput!) {
  updateUser(data: $data) {
    uuid
    updatedAt
    email
    isActive
  }
}
{
  "data": {
    "isActive": // boolean
    "uuid": "" // user uuid
  }
}

Set user Role​

To set a user role, you need to provide the user uuid and the role uuid.

Roles query​

query AllRoles {
  findAllRoles {
    items {
      uuid
      name
    }
  }
}

Update user role​

mutation UpdateUser($data: UpdateUserInput!) {
  updateUser(data: $data) {
    uuid
    email
    role {
      name
    }
    updatedAt
  }
}
{
  "data": {
    "role": {
      "uuid": "" // role uuid
    },
	"uuid": "" // user uuid
  }
}
